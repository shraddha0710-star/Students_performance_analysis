# 📊 Student Performance Analysis

## 📌 Overview
Student Performance Analysis is a data analytics project that evaluates student marks, attendance, study hours, assignments, and participation data. It converts raw data into meaningful insights using dashboards, KPI cards, and charts.

---

## 🎯 Objectives
- Analyze overall student performance  
- Compare attendance with marks  
- Study impact of study hours on results  
- Identify top-performing students  
- Detect weak areas for improvement  
- Present insights visually  

---

## 🛠️ Tools Used
- Microsoft Excel  
- PowerPoint  

---

## 📈 Dashboard Features
- 📌 KPI Cards  
- 🏆 Top 10 Students by Marks  
- ✅ Pass vs Fail Analysis  
- 📚 Subject-wise Average Marks  
- 👩‍🎓 Gender-wise Distribution  
- 💡 Final Insights & Recommendations  

---

## 🔍 Key Insights
- Total Students: 100  
- Average Attendance: 81.02%  
- Average Marks: 69.3  
- Pass Rate: 82%  
- Strong subjects: English & Computer  
- Science needs improvement focus  

---

## ✅ Conclusion
This project shows how student data can help improve academic planning and decision-making through clear visualizations and insights.
